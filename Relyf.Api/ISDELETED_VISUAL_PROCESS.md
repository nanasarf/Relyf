## ?? Safety & Rollback

```
MIGRATION SAFETY:

? Idempotent
   Can run script 10 times ? same result
   No errors if column already exists

? Non-destructive
   Only ADDS column, doesn't delete/modify data
   All existing rows unaffected

? Default Values
   IsDeleted = 0 (not deleted) by default
   Existing rows treated as "not deleted"

? Reversible
   DROP COLUMN IsDeleted if needed
   (Shouldn't need to - very safe)

? Zero Downtime
   No lock on table (ADD COLUMN is fast)
   Users can query while adding column
```
