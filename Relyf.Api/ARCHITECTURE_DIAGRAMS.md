## Notes

1. **Two Idea Types**: Projects can reference EITHER `IdeaId` (community) OR `AiIdeaId` (AI), but typically not both
2. **User Isolation**: All queries filtered by UserId to ensure data isolation
3. **Soft Deletes**: Records marked IsDeleted=1 rather than hard deleted
4. **Idempotent Queries**: Ownership checks prevent accidental data leaks
5. **Performance**: Indexes on UserId ensure fast queries even with large datasets
